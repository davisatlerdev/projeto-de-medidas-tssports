# Projeto-de-Medidas---TS-Sports

Sobre o Projeto

Este projeto foi desenvolvido para automatizar a etapa crítica de tradução de medidas na TS Sports. Antes realizado de forma manual, o processo de converter as dimensões corporais dos clientes para a grade de modelagem industrial agora é executado por um motor de regras em C#, garantindo que cada peça mantenha a integridade do design original e os padrões de conforto da marca.

Funcionalidades:

- Entrada de Dados: Coleta de medidas antropométricas do usuário.
- Processamento de Regras: Algoritmo de conversão baseado nos parâmetros técnicos da TS Sports.
- Output de Modelagem: Geração das medidas finais prontas para a equipe de corte/confecção.
- Margens de Segurança: Ajustes inteligentes de folga (folga de vestir) integrados ao cálculo.

Tecnologias:

- Linguagem: C#
- Framework: .NET CLI
- Paradigma: Orientação a Objetos (POO)
